EUVRAIN

Euvrain is my first personal website. This interactive website showcases a modern, Y2K-inspired design Madlib generator and a 'Guess the Number' game, both built using HTML, CSS, and JavaScript. The Madlib generator allows users to create short funny and engaging stories by filling in blanks with their own words. The 'Guess the Number' game challenges players to guess a randomly generated number with a celebratory confetti animation for correct guesses. I also feature some 3D characters I made on Blender.
