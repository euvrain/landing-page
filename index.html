IRIS

Project Type: Supervised Machine Learning · Multi-class Classification Tech Stack: Python, pandas, scikit-learn, matplotlib Description: This project uses the classic Iris dataset to classify flowers into three species based on petal and sepal measurements. It’s a simple yet powerful introduction to machine learning, allowing me to explore model training, evaluation, and data visualization.
  • Trained a multi-class classifier using scikit-learn
  • Visualized decision boundaries and feature importance
  • Evaluated performance with accuracy and a confusion matrix
  • Built a clean pipeline from preprocessing to prediction
Takeaway: This was my very first ML model — simple, elegant, and a great way to understand the full machine learning workflow from data to decision. And yes, it actually works.

EUVRAIN

Project Type: Interactive Web Design · Frontend Development Tech Stack: HTML, CSS, JavaScript, Blender Description: Euvrain is my first personal website, featuring a playful Y2K-inspired aesthetic and interactive web components. It includes:
  • Madlib Generator: Users create short, funny stories by filling in blanks with their own words.
  • Guess the Number Game: A number guessing challenge with a celebratory confetti animation on success.
  • 3D Art Showcase: A gallery of original 3D characters modeled and rendered in Blender.
Takeaway: This project helped me learn the basics of JavaScript interactivity, user experience design, and integrating custom 3D assets into a website. It’s quirky, creative, and uniquely mine.
